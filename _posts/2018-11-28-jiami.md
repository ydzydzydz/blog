`head.html`下`<meta charset="UTF-8"/>`后加入如下代码：
```
<script>
    (function(){
        if('{{ page.password }}'){
            if (prompt('请输入文章密码') !== '{{ page.password }}'){
                alert('密码错误');
                history.back();
            }
        }
    })();
</script>
```
需要加密的文章里加进`password: 你要设的密码`

```
---
title: 文章标题
layout: default
password: 123456
---
正文......

```
