**URL Scheme**

动作:

- 从选定的配置开始。
```
surge:///start
```

- 停止当前会话。
```
surge:///stop
```

- 使用所选配置启动或停止。
```
surge:///toggle
```

- 从URL安装配置。URL应进行编码。
```
surge:///install-config?url=x
```


选项:

动作完成后自动关闭Surge。（不能与install-config一起使用）

```
autoclose=true
```

示例:
```
surge:///toggle?autoclose=true
```
---

**x-callback-url**

Surge3.4支持x-callback-url。URL scheme 是`surge`，可用动作`start`,`stop`和`toggle`.
