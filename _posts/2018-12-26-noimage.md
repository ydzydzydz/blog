# 修改配置

打开Surge3,在`文本编辑模式`下对应字段复制代码如下:

### [URL Rewrite]

```
(.*)\.(jpg|bmp|gif|ico|pcx|jpeg|tif|png|raw|tga) - reject
```

### [MITM]

hostname =
```
*
```
